初始化配置

git config --global user.name "名称"
git config --global user.email 邮箱   #不一定要真实
git config --global credential.helper store   #保存
git config --global --list   #查看信息

新建仓库

仓库 : 目录
git init   #本地创建
git clone url   #克隆github 上的

添加和提交文件

git status   #查看仓库的状态
git add 文件名.txt   #添加到暂存区
           #可使用通配符, 例如: git add *.txt
           #也可以使用目录, 例如: git add .

回退版本

git reset
